# Proyecto Backend con NestJS

> [!IMPORTANT]
> Trabajo académico desarrollado en el marco de los **Cursos Cortos de Posgrado – UPEA**, aplicando tecnologías modernas para el desarrollo backend.

## Descripción
Este proyecto utiliza **NestJS**, un framework progresivo de **Node.js**, orientado a la construcción de aplicaciones backend **escalables, estructuradas y mantenibles**, basado en **TypeScript** y una arquitectura modular.

NestJS permite aplicar buenas prácticas de desarrollo de software, siendo una herramienta relevante tanto en el ámbito académico como profesional.

## Objetivo del Trabajo
Aplicar los conceptos fundamentales del desarrollo backend mediante el uso de NestJS, fortaleciendo las competencias técnicas adquiridas en los **cursos cortos de posgrado**.
